Sales Dashboard Analysis

An interactive Excel dashboard project built to analyze sales performance, profitability, and units sold across regions, products, and sales representatives.

📌 Project Overview

This project transforms raw sales data into a clear and interactive dashboard using Microsoft Excel. It helps businesses track key performance indicators (KPIs), identify top-performing products, monitor regional revenue, and evaluate salesperson contributions.

🛠 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Data Visualization
📊 Key Features
Interactive dashboard for sales analysis
Region-wise sales performance tracking
Product-wise revenue and profit analysis
Sales representative performance insights
Units sold comparison across categories
KPI tracking:
Total Sales
Total Profit

Units Sold
